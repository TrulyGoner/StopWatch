# Stopwatch

Приложение секундомер, построенное с использованием **React**, **TypeScript** и **Vite**. 
Приложение позволяет:
- Создавать несколько независимых секундомеров
- Запускать, паузировать и возобновлять отсчет времени
- Очищать и удалять секундомеры
- Отслеживать время в формате часов:минут:секунд

## Использованные технологии
- **React** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **Vite** - быстрый инструмент сборки
